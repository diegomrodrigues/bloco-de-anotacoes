Utilizando o Bloco de Anotações
Este software é um editor de anotações para arquivos de texto, ou arquivos de código fonte de programação.
A interface simples e fácil de ser utilizada, onde as linhas que aparecem, sendo numeradas sequencialmente, facilitando a visualização do texto, ou do código apresentado.


Menu Arquivo
  • Novo: Criar um novo arquivo
  • Abrir: Abrir um arquivo existente
  • Salvar: Salvar as alterações no arquivo atual
  • Salvar como: Salvar as alterações em um arquivo com outro nome
  • Sair: Fechar o Bloco de Anotações
Menu Ajuda
  • Conteúdo da Ajuda: Exibir uma janela onde serão exibidas as informações de ajuda
  • Sobre: Exibir a equipe de desenvolvimento, junto com a empresa e com a versão do Bloco de Anotações
